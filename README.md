## Description

Class library for use with MGM Insights website 
[https://mgminsights.rhtservices.net](https://mgminsights.rhtservices.net)

The resource classes within the library, are to be used with C# applications that 
call the MGM Insights Website. 

## Installation 

Using the terminal, navigate to your project's application directory. Then run the 
commands below.

```bash
dotnet add package Almostengr.MgmInsights.Common
dotnet restore
```

The package will download and install.
